== Description ==

Creates a XML feed from download-manager content. The XML feed can be used for creating an app store like clients.
== Installation ==

1. Upload `feed.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
